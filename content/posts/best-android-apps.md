## Videoder

<img src="https://raw.githubusercontent.com/videoder/main/master/public/images/toolbar_icon.png" alt="Videoder" width="120" />

### Features

- Download/Play Audio/Videos from almost any website
- Officially Supported Websites include:
  1. YouTube
  2. Facebook
  3. Instagram
  4. Twitter
  5. DailyMotion
  6. SoundCloud
  7. Vimeo
  8. IMDB
  9. and many more
- Built-in Web-browser
- Download in any Quality you like
- Download to any Folder you like (even on external memory-card/flash-drive)
- Copy Link Detection (When you copy a link to a video/audio, it shows a button you can press to start downloading the file)
- Share Link to Download Audio/Video file (no need to open Videoder just to download a video from YouTube)
- Auto-apply Album Art and other infromation to audio files
- Multiple Downloads at once (you can choose how many)
- Multiple threads per download (you can choose how many)
- Resume Interrupted Downloads
- Night Mode
- Theming (Primary/Accent Color)

##### Download/Install: [Visit Download Page](https://www.videoder.com/download/videoder-for-android)

## F-Droid

<img src="https://f-droid.org/repo/icons-640/org.fdroid.fdroid.1013050.png" alt="F-Droid" width="120" />

### Features

- Very lightweight

- Store for Open Source and Free Apps

- Apps in this store respect your privacy

  No personalized ads (most of the apps have no ads at all)

  No tracking

  No telemetry

- All apps in this store are open source

- All apps in this store are Free of cost (no in-app purchases either)

- Most of the apps are free software (free as in freedom)

  It means you can get their code, see exactly how they work, then change them and redistribute them with another name, without any worries about copyright laws. You have the freedom to use anything which is free software (free as in freedom) in any way you like. It is your legal right.

- You can add third-party Application Sources (where this store gets applications from)

##### Download/Install: [Direct Download](https://f-droid.org/F-Droid.apk)

## Termux

<img src="https://play-lh.googleusercontent.com/m3oqSZCwmitiZ-Im-CQu_rqT5eLHilOp5IudBynv3COJUumFzuQaP2dgTDxRL_03f4x2=s180" alt="Termux" width="96" />

This App is only for technical people, people who would like to learn/use Linux on their android device, students of Computer Science, or the people who want to learn hacking.

### Features

- Full fledged Linux Terminal on Android

- Most of the command-line linux apps are available to install including

  ssh, ftp (both server and client)

  dash, bash, zsh, fish command-line shells

  nano, vim, emacs text editors

  archiving utilities like 7z, zip, tar, xz, gz, p7zip, etc.

  compilers/interpreters like clang(c,c++), python, golang(go), ldc(d), etc.

  and much more

- APT package manager

- Can run complete Linux Operating System with Graphical Environments with the help of [VNC Viewer](https://play.google.com/store/apps/details?id=com.realvnc.viewer.android) and [AnLinux](https://f-droid.org/en/packages/exa.lnx.a)

##### Download/Install: [Play Store](https://play.google.com/store/apps/details?id=com.termux) | [F-Droid](https://f-droid.org/en/packages/com.termux/) (recommended)
